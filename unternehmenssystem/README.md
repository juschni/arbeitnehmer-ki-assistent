# Juschni Unternehmenssystem

**Status:** Sprint 0 aktiv  
**Zweck:** Single Source of Truth für Aufbau, Marke, Methodik, GPTs und spätere Produkte.

## Arbeitsmodus

**Build → Test → Lernen → Verbessern.**

Keine Theorie ohne Artefakt. Jede Arbeitseinheit endet mit einem nutzbaren Ergebnis.

## Struktur

| Bereich | Zweck |
|---|---|
| `00_dashboard` | zentrale Übersicht |
| `01_charta` | unveränderbarer Kern |
| `02_marke` | Marken-DNA, Sprache, Positionierung |
| `03_methodik` | Analyse- und Qualitätslogik |
| `04_gpts` | GPT-Konfigurationen |
| `05_testing` | Testfälle, Feedback, Auswertung |
| `06_roadmap` | nächste Etappen |
| `07_entscheidungen` | Entscheidungslog |
| `99_inbox` | Parkplatz für Ideen und Rohmaterial |
